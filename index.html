Variables aleatorias

  • Primero, una variable aleatoria no es un variable sino una función.
  • Se define sobre un espacio de probabilidad $(\Omega, \mathcal{F}, P)$.
  • Es decir, una variable aleatoria $X$ es una función sobre $\Omega$ hacía los números reales $\mathbb{R}$.

Variable aleatoria

Definición

Una variable aleatoria $X$, en un espacio de probabilidad $(\Omega, \mathcal{F},P)$, es un mapeo de $\Omega$ a $\mathbb{R}$, es decir, $X:\Omega \to \mathbb{R}$

$\Omega$ es el dominio de la variable aleatoria y $\mathbb{R}$ su rango.

Variable aleatoria

Variables aleatorias

  • Dependiendo de los valores del rango, la variables aleatorias pueden clasificarse como; discretas, continuas y mixtas
  • Usualmente, las variables aleatorias se describen con mayúsculas como $X,Y,Z, W, V$, etc.
  • Lo valores de una variable aleatoria se nombran con letras minúsculas, e.g., $x,y,z,w,v$.

Variables aleatorias

Ejemplos:

  • El experimento consiste en tirar un par de dados. Se define la variable aleatora $X$ como la suma de valores que aperecen en los dados.
  • Se tira una moneda al aire. Se define la variable aleatoria como: $$ X = \begin{cases} 10 & \mbox{aparece aguila}\\ -10 & \mbox{si aparece sol} \end{cases} $$

Variables aleatorias

Condiciones de existencia:

Para que $X$ sea considerada una variable aleatoria se tiene que cumplir que:

  • El conjunto $\{X \le x\}$, es un evento.
  • La probabilidades de $\{X=+\infty\}$ y $\{X=-\infty\}$ son:
    $$P(X=+\infty) = P(X=-\infty) = 0$$

Variables aleatorias

CDF, PDF y PMF

Las variables aleatorias quedan especificadas por:

  • La función de distribución acumulativa (CDF).
  • Si $X$ es continua, por la función de densidad de probabilidad (PDF).
  • Si $X$ es discreta, por la función de masa de probabilidad (PMF).

Función de distribución acumulativa

Definición

La función de distribución de una variable aleatoria $X$ es la función definida como: $$ F_X(x) = P\left\{ X \le x\right\}, $$
definida para cualquier número $x$ de $-\infty$ a $+\infty$.

Función de distribución acumulativa

Propiedades de la CDF:

  • $0 \le F_X(x) \le 1$
  • $F_X(-\infty) =0 $ y $ F_X(+\infty)=1$
  • Si $x_1 < x_2$ entonces $F_X(x_1) \le F_X(x_2)$
  • $P(a < X \le b) = F_X(b) -F_X(a) \;$ $a< b$
  • $P(X>a) = 1-F_X(a)$

CDF: Ejemplos

Sea $X$ una variable aleatoria con distribución: $$ F_X(x) = \begin{cases} 0 & x < -1\\ 1/4 & -1 \le x < 1\\ 1/2 & 1 \le x < 3\\ 3/4 & 3 \le x < 5\\ 1 & x \ge 5 \end{cases} $$ Calcular $P(-1 < X \le 1.5)$ y $P(X\ge 1)$

Solución: Recordemos que; $P(a < X\le b) = F_X(b)-F_X(a)$, por lo tanto: $$ P(-1 < X \le 1.5) = F_X(1.5)- F_X(-1) = 1/2- 1/4= 1/4 $$

Solución: Recordemos que; $P(X\ge b) = 1-F_x(b)+P(X=b)$, por lo tanto: $$ P(X \le 1) = 1-F_X(1)+P(X=1) = 1-1/2+1/4 = 3/4 $$

CDF: Ejemplos

Sea $X$ una variable aleatoria con $F_X(x)$: $$ F_X(x) = \begin{cases} 0 & x < 1\\ 2\left(1- \frac{1}{x}\right) & 1 \le x < 2\\ 1 & x \ge 2 \end{cases} $$ Calcular $P(-1 < X \le 1.5)$

Solución: Recordemos que $P(a < X \le b) = F_X(b)-F_X(a)$ , por lo tanto: $$\begin{align} P(-1 < X \le 1.5) = & F_X(1.5)- F_X(-1)\\ =& 2/3 - 0 = 2/3 \end{align}$$

Función de densidad (PDF)

Definición

Sea $F_X(x)$ la función de distribución de una variable aleatoria $X$, la función de densidad de $X$ denotada por $f_X(x)$ está definida por: $$ f_X(x) = \frac{dF_X(x)}{dx}, $$
donde $x \in \mathbb{R}$.

Función de densidad (PDF)

Propiedades de la PDF:

  • $f_X(x)\ge 0$
  • $\int_{-\infty}^{+\infty}{f_X(x)}\, dx =1$
  • $P(a < X \le b) = \int_{a}^b{f(x)\, dx}$
  • $F_X(b) = \int_{-\infty}^b{f(u)\, du}$

PDF: Ejemplos

La función de densidad de una variable aleatoria está dada por: $$ f_X(x) = \begin{cases} x & 0 < x < 1\\ 2-x & 1 \le x < 2\\ 0 & \mbox{en otro caso} \end{cases} $$ Calcular $P(0.2 < X \le 0.8)$

Solución: Recordemos que $P(a < X \le b) = \int_{a}^b{f_X(x)\, dx}$ , por lo tanto: $$\begin{align} P(0.2 < X \le 0.8) = & \int_{0.2}^{0.8}{x\, dx} = \frac{x^2}{2} \biggr\rvert_{0.2}^{0.8}\\ =& \frac{1}{2}\times\left((0.8)^2-(0.2)^2\right) = 0.3. \end{align}$$

Función de masa de probabilidad (PMF)

Definición

Sea $X$ una variable aleatoria discreta con valores $x_1, x_2, ... x_n$, la función de masa de probabilidad de $X$ está dada por: $$ p_X(x_i) = P(X=x_i) = F_X(x_i)-F_(x_{i-1}), $$
donde $x_i < x_j$ si $i < j$.

Función de masa de probabilidad (PMF)

Propiedades de la PMF:

  • $0 \le p_X(x_i)\le 1$
  • $p_X(x)=0$ si $x\ne x_k \, (k=1,2,\ldots, n)$
  • $\sum_k{p_X(x_k)}=1$
  • $F_X(x) = \sum_{x_k \le x}{p_X(x_k)}$
